🤖 Ollama 대화형 인터페이스

FastAPI 기반 통합 AI 서비스 플랫폼

🟢 서비스 상태: 정상 운영 중

💬 AI 대화

Ollama 모델과 실시간 대화를 통해 질문하고 답변을 받을 수 있습니다. 다양한 AI 모델을 지원합니다.

📈 주식 정보

실시간 주식 데이터 조회, 가격 분석, 종목 검색 등 한국 주식 시장 정보를 제공합니다.

🌤️ 날씨 정보

도시별 실시간 날씨 정보, 예보, 대기질 정보를 제공합니다.

🔍 웹 검색

키워드 기반 웹 검색, 웹페이지 내용 가져오기 등 인터넷 정보 검색 서비스를 제공합니다.

📁 파일 관리

파일 읽기, 쓰기, 검색, 목록 조회 등 파일 시스템 관리 기능을 제공합니다.

🗄️ 데이터베이스

사용자 관리, 노트 관리, SQL 쿼리 실행 등 데이터베이스 관리 기능을 제공합니다.

📚 API 엔드포인트

💬 채팅 API

POST /api/chat - AI 모델과 대화

GET /api/chat/models - 사용 가능한 모델 목록

GET /api/chat/sessions - 세션 관리

📈 주식 API

GET /api/stocks/{code} - 주식 정보 조회

GET /api/stocks/search/{keyword} - 종목 검색

GET /api/stocks/{code}/price - 가격 데이터

🌤️ 날씨 API

GET /api/weather/{city} - 날씨 정보

GET /api/weather/forecast/{city} - 날씨 예보

GET /api/weather/air-quality/{city} - 대기질

🔍 웹 검색 API

GET /api/web/search - 웹 검색

GET /api/web/fetch - 웹페이지 가져오기

GET /api/web/trending - 인기 검색어

📁 파일 API

GET /api/files - 파일 목록

GET /api/files/{path} - 파일 읽기

POST /api/files/{path} - 파일 쓰기

🗄️ 데이터베이스 API

GET /api/db/users - 사용자 목록

POST /api/db/notes - 노트 추가

POST /api/db/query - SQL 쿼리 실행
